# NextJS with Redux and Redux-Saga (Docker)

## Install

```
npm install
```

## Run

```
npm run dev
```

## Build

```
npm run build
```

## Run (Production)

```
npm start
```

## Docker Build

Mac, Linux

```
./scripts/prod.sh
```

Windows

```
./scripts/prod.bat
```

## Docker Run

```
docker run -p 3000:3000 nextjs-docker
```
